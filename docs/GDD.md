# 斩币（暂名）游戏设计文档

> 屏幕就是战场，你的手就是刀。用划砍守住底部平台，让金币堆积合成，换取更强的斩击能力。

---

## 1. 游戏概述

### 1.1 核心基因

| 来源 | 贡献 |
|---|---|
| 切水果 | 划砍手感、刀光碰撞、即时反馈 |
| 合成大西瓜 | 产物物理堆叠、等级合成、连锁反应 |
| 吸血鬼幸存者 | 波次节奏、增益成长、长局压迫感 |

### 1.2 屏幕结构

```
┌─────────────────────────┐
│                         │
│   敌人从上方/侧方涌入    │
│   玩家在此划砍           │  ← 上半屏：战斗区
│                         │
├─────────────────────────┤
│  ● ◆ ● ●  ◆            │
│ ● ● ● ● ● ● ◆ ●       │  ← 下半屏：合成区
└─────────────────────────┘  底部平台（铸币台/祭坛/砚台）
```

### 1.3 核心循环

```
敌人涌入 → 划砍切掉 → 产物洒落堆积
    ↓                        ↓
漏过的敌人吃产物        同类 3 个接触 → 合成
    ↓                        ↓
紧迫感、补切冲动      ┌─ 留着继续合成更高级
                      ├─ 消耗换增益（主动决策）
                      └─ 首次达到新稀有度 → 免费增益
                                ↓
                         恢复战斗，平台空了一块
                         冲动去切更多敌人补回来
```

### 1.4 失败条件

平台上的产物被吃光，游戏结束。不是瞬间死亡，是积累被彻底清空。

## 2. 划砍系统

### 2.1 基础机制

- 手指划过的轨迹 = 刀，实时碰撞检测
- 划过哪里就造成伤害，不需要画完才触发
- 划得越快，刀光越宽、伤害越高

### 2.2 切中反馈

- 敌人沿切割方向裂开飞出
- 弹幕被切断炸成光点
- 连切多目标音效叠加（音调递升）

### 2.3 分区交互规则

| 操作 | 上半屏（战斗区） | 下半屏（合成区） |
|---|---|---|
| 快速划砍 | 切割敌人，造成伤害 | 切杀落地敌人（不掉产物）；击飞碰到的产物 |
| 长按产物 1s | — | 消耗产物，弹出三选一增益（圆形进度条确认） |
| 不操作 | 敌人继续下落 | 落地敌人持续吃产物（1枚/秒） |

### 2.4 刀光与产物的物理交互

- 刀光碰到产物会沿切割方向弹开
- 力度与刀速成正比：轻划微推，大力快划弹飞
- 玩家可利用此机制主动拨动产物，将同类归拢以加速合成
- 与长按消耗不冲突：短接触 = 击飞，长接触 1s = 消耗换增益

## 3. 产物系统

### 3.1 合成规则

- **3 合 1**：3 个相同产物接触时自动合成为 1 个更高级产物
- 合成后产物体积更大，可能挤压周围产物触发连锁合成

### 3.2 分支合成树

合成结果不是固定的下一级，而是从多个可能的分支产物中**随机产出一个**：

```
               ┌─ 产物A1 ─┬─ 产物A1a
               │           └─ 产物A1b
      ┌─ 产物A ┤
      │        └─ 产物A2 ── ...
基础 ─┤
      ├─ 产物B ─┬─ 产物B1 ── ...
      │         └─ 产物B2 ── ...
      └─ 产物C ─┬─ 产物C1 ── ...
               └─ 产物C2 ── ...
```

- 同一局游戏拿到的产物组合独一无二
- 合成路径越深，产物越稀有，分支越多样
- 首次发现新产物时屏幕角落弹出"新发现！"提示（不暂停），记入图鉴

### 3.3 分支权重

- 默认各分支等概率
- 受遗物影响：流派遗物会将权重大幅偏向某个分支方向（约 70%）
- 形成"流派 build"体验

### 3.4 分支数量控制

- 第 1→2 级：2-3 种可能产物（前期集中，保证能合成）
- 第 2→3 级：3-4 种
- 第 3→4 级：4-5 种
- 越深层分支越多，但前期保持集中避免碎片化

### 3.5 物理模拟参数（参考值）

| 参数 | 值 | 说明 |
|---|---|---|
| 重力加速度 | 9.8 m/s² | 自然下落感 |
| 摩擦系数 | 0.3-0.5 | 影响滚动距离 |
| 弹性系数 | 0.2-0.4 | 控制弹跳高度 |
| 质量比 | 逐级递增 | 越高级越重越稳定 |

### 3.6 连锁合成

- 合成产生的新产物体积更大
- 体积增大可能挤到旁边的同级产物
- 触发下一级合成 → 连锁反应
- 连锁合成时视觉 + 音效 + 物理爆发力三重反馈
- 合成时有轻微"爆"的力推开周围产物，增加连锁概率

## 4. 敌人系统

### 4.1 基础敌人

**直落型**
- 从屏幕顶部直线下落
- 轨迹可预测，速度随波次递增
- 密集出现时像一面帘子往下压

**侧抛型**
- 从屏幕左/右侧边 1/2 处往上抛出
- 抛物线顶点在屏幕顶端，然后下落
- 顶点时速度最慢（物理真实），是最佳切割窗口
- 水平位移让落点更难预判

```
  直落型          侧抛型

  ▼               ╭──╮
  ▼              ╱    ╲
  ▼             ╱      ╲
  ▼            ╱        ╲
  ▼           ●(入口)    ╲
              侧边1/2处   ╲
```

### 4.2 特殊敌人：炸弹型

- **不可切割**，需要玩家在划砍中精准避开
- **误切惩罚**：从平台所有现存产物中真随机选一枚消耗（无增益），可能吃铜币也可能吃宝石
- **漏到平台的奖励**：根据当前波次/进度，随机变成一枚对应等级范围的产物
- 反转了核心操作指令——普通敌人必须切，炸弹必须放过

### 4.3 漏过的敌人行为

- 落在平台上，落在哪里就吃哪里的产物
- 吃产物速度固定：**每秒 1 枚**，无论产物等级
- 高级产物被吃 = 大量基础产物价值损失
- 漏过的敌人**可以被补杀**（在下半屏划砍），但补杀不掉落产物
- 漏过的敌人需有明显视觉标记（变灰/发红），与产物在视觉层上区分

### 4.4 敌人掉落规则

- 敌人死在哪里，产物就从那个位置洒落
- 有物理重力，互相碰撞，自然堆积在平台上
- 不同品质的敌人有不同的总价值范围，在范围内随机决定掉落的产物组合

## 5. 增益系统

### 5.1 获取途径

**途径一：主动消耗（随时可用）**
- 波次间隙或战斗中，长按平台上任意产物 1 秒
- 圆形进度条确认，防止误触
- 弹出三选一增益面板，选择后该产物被消耗
- 增益稀有度/强度与被消耗产物的等级挂钩

**途径二：稀有度里程碑（被动触发，免费）**
- 首次合成出新的稀有度等级（深度层级）时，游戏暂停
- 弹出免费三选一增益，不消耗任何产物
- 无论是哪个分支产物，只要是首次到达该深度就触发
- 整局约触发 4-6 次（等于深度层数 - 1），每次都是真正的里程碑
- 同时起教学作用：让玩家理解"合成 = 力量"

### 5.2 增益方向（示例）

- 刀光宽度增加
- 连切三刀触发爆炸
- 划出封闭曲线消除圈内弹幕
- 金币掉落偏向平台中央
- 连续不同切法触发元素爆发
- 击杀回复效果

### 5.3 两难决策

> 这枚高级产物，是留着继续合成更高级的，还是现在兑换一个强力增益？

消耗高级产物换增益 = 当前局变强，但失去了继续向上合成的可能性。

## 6. 遗物系统

### 6.1 定位

遗物是**最上层的 build 框架**——不给具体数值，而是决定这局游戏的流派方向。

```
第三层：遗物（被动定义流派）  — "我这局是什么 build"
第二层：增益（主动消耗强化）  — "我现在需要什么能力"
第一层：产物（资源/货币）     — "我有多少财富"
```

### 6.2 获取方式

**波次里程碑，前密后疏：**

| 波次 | 2 | 4 | 7 | 11 | 16 | 22 | 29 |
|---|---|---|---|---|---|---|---|
| 间隔 | 2 | 2 | 3 | 4 | 5 | 6 | 7 |
| 遗物 | #1 | #2 | #3 | #4 | #5 | #6 | #7 |

- 前 7 波拿到 3 个遗物，流派方向基本成型
- 中后期间隔拉长，每个遗物获取都伴随更大的波次压力
- 超过 30 波后的里程碑必须替换已有遗物

### 6.3 槽位

- 最多 **7 个**遗物槽位
- 满槽后获得新遗物必须替换已有遗物

### 6.4 遗物类型

**流派遗物（影响合成分支权重）**

每个流派遗物将合成分支的随机权重大幅偏向某个方向：

- 火系："熔炉之心"（70% 出火系）、"淬火印"（火系消耗增益+50%）、"燎原"（火系合成爆炸力更大）
- 冰系："霜降"（70% 出冰系）、"冻结核心"（冰系产物减速敌人吃产物）、"碎冰"（冰系连锁合成对敌人造成伤害）
- 雷系："引雷针"（70% 出雷系）、"超导"（雷系产物远距离合成）、"磁场"（雷系产物吸引同类缓慢靠拢）

**通用遗物（改变基础规则）**

- "宽刃"：刀光基础宽度 +30%，但更容易误切炸弹
- "吝啬鬼的护符"：敌人吃产物速度减半，但掉落产物也减半
- "重力透镜"：产物掉落偏向平台中央
- "赌徒的骰子"：取消流派权重全随机，但合成有 20% 概率跳一级
- "拾荒者"：炸弹漏到平台后变成的产物等级 +1

### 6.5 遗物协同（Combo）

- "熔炉之心" + "燎原" = 连锁流（火系多 + 爆炸力大 → 连锁暴增）
- "霜降" + "吝啬鬼的护符" = 囤积流（敌人吃产物极慢 → 安全积累）
- "引雷针" + "超导" = 散布流（远距离合成 → 不怕产物分散）

## 7. 局外养成系统

### 7.1 遗产货币

**来源：** 每局结束时，平台上残留的所有产物按等级折算为"遗产"点数。

| 产物等级 | 遗产价值 |
|---|---|
| 1 级 | 1 |
| 2 级 | 5 |
| 3 级 | 25 |
| 4 级 | 100 |
| 5 级 | 400 |
| ... | 递增 |

**设计张力：** 局内消耗产物换增益 = 当前变强但遗产更少；保留产物 = 当前较弱但死后遗产更多。

### 7.2 内容解锁（主线）

用遗产点数解锁新内容，加入局内随机池：

- **新遗物**：初始池 8-10 个，逐步解锁更多
- **新增益**：高级增益需要先解锁才会出现在选项中
- **新合成分支**：初始 2-3 条分支线，逐步解锁新分支
- **新敌人类型**：带护盾的、会闪避的、分裂型的等
- 解锁 = 加入随机池，不是直接拥有

### 7.3 永久微提升（辅线，有上限）

| 升级项 | 等级 | 效果 |
|---|---|---|
| 初始铜币 | 0/1/2/3 | 开局平台自带铜币数 |
| 刀光基础宽度 | +0%/+5%/+10%/+15% | 永久宽度提升 |
| 敌人吃产物减速 | -0%/-5%/-10%/-15% | 减缓吃产物速度 |
| 产物掉落偏移 | 0%/10%/20%/30% | 落点向平台中心偏移 |
| 遗物槽位 | +0/+1 | 从 7 变为最多 8 |

每项有明确满级上限（4 级），费用递增，不会无限膨胀。全部满级后遗产转为排行榜积分。

### 7.4 图鉴系统

- 记录发现的所有产物种类
- 不给数值奖励，纯收集展示
- 图鉴完成度可作为社交分享素材
- 新产物发现时角落弹出小提示，不暂停游戏

### 7.5 解锁节奏

| 局数 | 阶段 | 体验 |
|---|---|---|
| 1-3 | 新手期 | 解锁基础增益 + 1 个永久提升，"死了也有收获" |
| 4-10 | 探索期 | 解锁新分支/遗物，每局发现新内容 |
| 11-20 | 精通期 | 解锁高级敌人/稀有增益，开始研究 build |
| 20+ | 毕业期 | 大部分解锁，转向最高波次/分数挑战 |

## 8. 成就系统

### 8.1 阶梯型成就（跨局累积）

**斩击系列**
- "初试锋芒"（10）→ "百人斩"（100）→ "千人斩"（1000）→ "万人斩"（10000）

**连击系列**
- "连斩"（单划砍切中 3 个）→ "狂斩"（5 个）→ "神斩"（8 个）

**合成系列**
- "炼金入门"（50 次）→ "炼金大师"（300 次）→ "炼金传奇"（1500 次）

**存活系列**
- "顽强"（第 10 波）→ "不屈"（第 20 波）→ "永恒"（第 30 波）

**财富系列**
- "小有积蓄"（总价值 50）→ "腰缠万贯"（200）→ "富可敌国"（1000）

### 8.2 挑战型成就（单局达成）

**操作挑战**
- "完美防线"：连续 5 波不漏过任何敌人
- "刀刃之舞"：单局切到 0 个炸弹（至少通过 15 波）
- "精准外科医生"：在下半屏斩杀落地敌人时不击飞任何产物（至少 5 次）

**策略挑战**
- "苦行者"：不消耗任何产物换增益，通过 20 波
- "挥金如土"：单局消耗 10 个以上产物换增益
- "赌徒"：同一局拥有过 5 个不同流派的遗物

**极端挑战**
- "绝地反击"：平台只剩 1 枚产物时，重新积累到 20 枚以上
- "空手套白狼"：不升级任何永久属性的情况下通过 25 波

### 8.3 产物组合型成就（平台同时拥有特定组合）

**同系收集**
- "烈焰全家福"：同时存在火系 3 种不同产物
- "寒冰全家福"：同时存在冰系 3 种不同产物
- "万象归一"：每个流派至少 1 种产物同时存在

**跨系搭配**
- "冰与火之歌"：同时存在最高级火系和最高级冰系产物
- "三元素共鸣"：同时存在 3 种不同流派的 3 级以上产物
- "彩虹"：同时存在 7 种以上不同种类产物

**稀有组合**
- "双生"：同时存在 2 个相同的 4 级产物
- "至高三连"：同时存在 3 个相同的最高级产物
- "满堂红"：平台 15 个以上产物且无任何铜币

**隐藏组合**
- 条件不提前显示，特定产物相邻放置时触发
- 达成后才揭示名称和描述
- 社区讨论和攻略素材

### 8.4 成就奖励

- **称号**：白色（常见）→ 绿色（优秀）→ 蓝色（精英）→ 紫色（史诗）→ 金色（传说）
- **遗产点数**：阶梯型逐级递增，挑战型一次性大额
- 玩家可选择佩戴任意已解锁称号，金色称号在排行榜发光

### 8.5 成就的引导功能

前几个成就自然引导玩家发现机制：
- "第一次合成" → 教会合成
- "第一次消耗产物换增益" → 教会增益系统
- "第一次切到炸弹" → 教会炸弹规则

## 9. 世界观备选

### 方向 A：墨剑书境

- **设定**：一切发生在古卷之中，玩家是封印在卷轴里的剑灵，敌人是渗入的墨妖
- **刀光**：流动的墨痕
- **产物**：斩杀墨妖后凝结的"灵墨"，不同书体对应不同流派
- **美术风格**：水墨风 + 书法笔触，宣纸质感背景
- **优势**：强烈中国文化辨识度，海外新鲜感，视觉简洁性能开销小

### 方向 B：深渊铸币局

- **设定**：世界尽头悬崖上的铸币局，深渊中的贪婪之物攀爬夺回"欲望之源"
- **刀光**：灼热的金属弧线
- **产物**：铸造出的各类货币/宝物
- **美术风格**：暗黑哥特 + 金属光泽，深色背景配金铜高光
- **优势**：与"斩币"主题完美呼应，暗黑风对核心玩家有吸引力

### 方向 C：星陨祭坛

- **设定**：永不停歇的流星雨中，祭坛守护者用星刃切碎陨落的星兽
- **刀光**：星光尾迹
- **产物**：星晶碎片，同频共振融合
- **美术风格**：深空宇宙 + 霓虹光效，发光粒子和星光特效
- **优势**：视觉表现力最强，受众最广

**原型阶段**：使用几何形状 + 颜色区分，不依赖具体美术资源。
